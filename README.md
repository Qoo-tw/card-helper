# 第一個ChatGPT寫的程式
# card-helper 💳  
刷卡回饋快速判斷小工具（PWA）
卡片有: 大戶黑卡、永豐幣倍、星展ECO、匯豐匯賺
這是一個為了「少開 Excel、快速知道該刷哪張卡」而做的前端小工具。  
只要輸入 **店家 / 描述 + 金額**，系統就會根據你自訂的規則，自動判斷：

- 國內 / 國外
- 命中哪一張卡、哪一條回饋規則
- 預估可拿多少回饋

本專案為 **純前端、無後端**，資料完全由使用者自己維護，適合個人使用。

---

## ✨ 功能特色

- 🔍 **關鍵字比對店家**
  - 例如：`鮮乳坊 / Agoda / Spotify / Uber Eats`
- 🌏 **自動判斷國內 / 國外**
  - 依照 merchant 關鍵字對應設定
- 💳 **多張卡片、多條回饋規則**
  - 支援不同卡片、不同回饋 %
- 🧠 **命中規則自動選擇**
  - 只要有命中即可
  - 優先選擇回饋 % 較高的規則
- 📱 **PWA 支援**
  - 可加入 iPhone 主畫面
  - 離線可用（Service Worker）
- 🔐 **資料不外流**
  - 全部在瀏覽器端運算，不送出任何資料
